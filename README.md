# Features
- UPM Package
- fill floating
- fill per-sprite-segment
- fill per-vertex
- fill Handle (selected segment only)
- fill Handle Inverted (all except selected segment)
- skip offset
- intersecting (overlapping) segments
- sprite round
- rotate offset
- round gradient
- GC-free
- border / outline mode
